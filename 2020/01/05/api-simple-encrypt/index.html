简单的接口加密

Jan 05, 2020Skills点击

在前后端的数据传输中，为了保证数据安全等原因，我们需要对传输的数据进行加密。

流程

客户端发送请求前加密，服务端接收到数据后解密
服务端返回数据前加密，客户端接收到数据后解密

加密方式

加密方式采用 aes-128-cbc
客户端和服务端的key和iv要保持一致
