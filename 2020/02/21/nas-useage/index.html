群辉Nas的简单使用

Feb 21, 2020点击

最近因为电脑磁盘空间不够，考虑买移动硬盘，但移动硬盘每次使用时都需要插到电脑上，非常麻烦，所以干脆就购买了Nas。

简介

Nas（Network Attached Storage）网络附属存储，通俗来讲就是可以联网的存储装置。

购买原因

电脑磁盘太小512G，而我平时骑车会用GoPro拍摄一些视频，一次就会有几十个G。

我为什么不直接购买移动硬盘？原因很简单：使用不方便，每次使用都需要连接上电脑，非常麻烦。

Nas的优势：他可以通过网线与路由器相连，我们再使用电脑挂载上Nas的共享文件夹，就能实现随时可用。

配置

我购买的是群辉的DS418Play，配置如下：

  • 商品名称：群晖DS418play
  • 商品毛重：3.625kg
  • 商品产地：中国台湾
  • USB2.0：2个
  • 核心数：双核
  • USB3.0：2个
  • 操作系统：QTS
  • 内存：2G
  • 网口速率：千兆网口
  • Thunderbolt：2个
  • 网口：2个
  • 类别：家庭网络存储
  • 材质：PVC
  • 处理器：Intel
  • 规格：四盘NAS

硬盘我选用的是4块酷狼2T的Nas专用盘，组了RAID5

使用

挂载到电脑

我们可以在Nas上建立共享文件夹，然后挂载到电脑上使用。

外网访问

群辉Nas的Quickconnect可以让你在外网访问家中的Nas。

群辉的套件中心提供了许多套件（官方第三地方），可以根据需求下载。

下面是一些我使用到的功能：

图片

使用 Moments 套件，提供了安卓+iOS+Web APP，可以实现手机照片自动同步到Nas。

iOS

Web

影片

使用 Plex 套件，可以自动为影片匹配简介、海报、字幕，Plex还支持多端，你可以在手机、电脑、电视上观看Nas中的影片

文件

使用Ds File可以在手机上浏览Nas中的文件。

其他

Nas还有其他很多玩法，比如Docker、Gitlab等等。
